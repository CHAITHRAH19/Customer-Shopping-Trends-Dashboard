## Overview

The **Customer Shopping Trends Dashboard** is an interactive data visualization project built using **Tableau** to analyze and interpret customer buying behavior in the retail sector. The dashboard transforms raw retail data into actionable insights, helping stakeholders make informed decisions related to inventory management, marketing strategies, and customer segmentation.

This project is based on a publicly available **Kaggle dataset** and was developed as part of the module **"Visualization and Story Telling using Tableau" (2025)**.

---

## Objectives

The main objectives of this project are:

* Analyze customer purchasing behavior across seasons
* Identify spending patterns by gender and region
* Perform customer segmentation based on demographics and expenditure
* Understand revenue contribution by product category and geography
* Support data-driven retail decision-making

---

## Dataset

* **Source:** Kaggle ‚Äì Customer Shopping Trends Dataset
* **Link:** [https://www.kaggle.com/datasets/iamsouravbanerjee/customer-shopping-trends-dataset](https://www.kaggle.com/datasets/iamsouravbanerjee/customer-shopping-trends-dataset)

### Key Data Attributes

* **Customer Demographics:** Customer ID, Gender, State (USA)
* **Transaction Details:** Purchase amount, payment method, shipping type
* **Product Information:** Category (Clothing, Accessories, Footwear, Outerwear), size, color
* **Temporal Data:** Seasons (Spring, Summer, Fall, Winter)
* **Additional Metrics:** Ratings, discounts, promo codes, subscription status

---

## Data Preprocessing

The dataset was cleaned and prepared before visualization:

* Removed duplicate records
* Handled missing values
* Standardized data formats
* Created calculated fields (discount impact, revenue metrics)

This ensured accuracy and consistency in all dashboard visualizations.

## Dashboard Features

The Tableau dashboard includes the following components:

### 1. Customer Segmentation

* Scatter plot showing spending behavior by customer and gender
* Identifies high-value and low-value customer segments

### 2. Seasonal Purchase Trends

* Bar charts displaying product category performance across seasons
* Helps anticipate demand and plan inventory

### 3. Top Purchases by Gender

* Gender-based comparison of spending on key product categories
* Supports targeted marketing strategies

### 4. Geographic Distribution

* Bubble map showing purchase volume by U.S. state
* Highlights high-performing and underperforming regions

### 5. Revenue Contribution Analysis

* Revenue breakdown by category, gender, and location
* Identifies the most profitable segments

### 6. Storyboard

* A guided narrative combining key insights for executive decision-making
